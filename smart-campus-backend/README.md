# Smart Campus Backend (WIP)

Folder ini disiapkan untuk backend/publisher yang akan mengirim data ke MQTT broker.

## Tugas backend (high-level)

- Publish payload JSON ke topic `campus/...` sesuai kontrak di `../smart-campus-dashboard/README.md`.
- Menjaga konsistensi field `roomId`, `roomName`, dan `timestamp`.
